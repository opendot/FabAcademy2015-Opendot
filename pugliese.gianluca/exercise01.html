Web page

I have started by studying the HTML and CSS code in order to create this website; I've used Brackets to write the code, I've created a custom version CMS starting from A Template of Massimo Menichinelli for Bootstrap and have changed many parameters in the HTML and CSS files.

The most difficult part was to read the whole code, learn how it works and then customize it in the right way.

A big help came from Internet: there are a lot of useful websites where you can learn HTML and CSS.

For example, I had to force the color for one type of link by adding a part of CSS code in the HTML page directly. This was because I could not find the correct way to substitute them all at the same time.

	<p>
	<!-- LINK COLOUR FORCED... To Be Solved -->   
      
      <style>
         a{
              color:#ff6200;
          }
      </style>
	</p>
