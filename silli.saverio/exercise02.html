WEEK 2

02. Computer-aided Design

Draw sketches, vector/CAD drawings and 3D models


As an Architect, I already knew many of the softwares, so I just kept using those I know better. First of all I drew a sketch freehand, then scanned it and put into photoshop to manage exposure levels and saturation. I saved for web to have a very small size file.

freehand sketch

I then used an architectural design software I know very well called Vectorworks to make technical drawings with quotes and saved it in SVG.

SVG file with quotes

I used Sketch-up to make a quick 3D model which I animated into a gif. To do so I moved the model by 30degrees every time and exported a jpg for every step, it took 12 jpg to see a complete rotation of the rotocasting machine and at the end I created a gif with an online tool called Gifmaker.


A complete rotation of 360 degrees

From sketch up I also exported the model in STL using Binary in the saving settings to reduce file size.
