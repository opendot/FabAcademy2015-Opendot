Computer Aided Design



The weekly assignment consist in experiment some CAD (computer aided design) 2d and 3d.



Blender



I found it a very useful software: it is open source, free and very light in comparing to the things it can make.
You can work in mesh or on the surface with some brushes.
It is really indicated for some artistic purpose, he had a nice rendering engine, but it is not simple to use, because of the high number of options and commands.
The interface is full of things and this means some confusion finding what you need.





I used blender in the composite lecture for editing the Captain America's shield, modifying directly the mesh to make the shield a "one" volume to be milled in 3d.

FreeCAD



This free software is most oriented to the mechanical and precision drawing.
It have some pre-compiled tool for architectural drawings and for naval design.
I used it for draw a first representation of the mechanical system of my robotic storage.
In the pics below you can see my attempt to draw a xy movement system.
It is for the select-head of my system, it will be placed in the back of storage drawers.

Inkscape



This is a free software who make vectorial file, used by most of the CAM program ( laser, cnc router, etc).
I use this tool for make a vector image of the h-bot system I will use on the robotic storage.
I make some analysis of the tension on the driving belt, for showing where them are pulled and where them are pushed.






Download the SVG file





ThinkerCad



I used a lot this web software for make some .stl file for 3D printing.
The interface is really user friendly and the measuring system is nice and fast.
The limitation is that you can make only simple models because the power is limited to the strength of the internet file and browser spec.
Despite this is very useful for some fast, simple and little 3D objects.
