Invention, Intellectual Property, and Income



This week we talk about the follow up on the project and how to protect/develop and share it whit others; start a plan to sell the project and crowfounding.


Invention & Intellectual properties

My final project is realized for be made in a Fablab, adjusted for the desired need, made of cheap and easy to find material.


For the fact I used a open source Firmware (grbl) i want to make my project open source too, so maybe someone interested can help me develop this project, make new feature, modify it , rebuild it... and so on.


For this reason:

Income

I don't know if this project can have a related income.
As soon as I publish some videos and picture some fiends ask if I can build it for them but i don't think in that way can be crow-founded or transformed in a market product.
maybe I can build some custom copy of this for some lab or someone who want as individual work but that's all.
I want to keep open-source all the improvements that I made, and I also be really honoured if someone will work on my project.
