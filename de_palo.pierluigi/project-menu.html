In this section I will try to summarize my progress with my Fab Academy Final Project : The Photogrammetric Fab Scanner.

This section includes the different development phases of the machine. During these stages, starting from the initial concept, I was able to create three different prototypes: each one of them was developed as a feasibility study, a theoretical and practical approach to version 1.0

The versions produced were:


→V0.1 // CONCEPT : the basic idea, the theoretical functional principles of the machines

→V0.2 // PHYSICAL CONCEPT TEST : the practical testing of the basic prototype that lacked structure, illumination. Any form of automation.

→V0.3 // INANIMATE VERSION : the structural prototype, that was illuminated, fitted with devices to simulate the movements of the machine and a Rotator Graduated Head to simulate the precise rotation of the article to be scanned.

→V0.4 // ANIMATE VERSION : a complete structural prototype complete with: autonomous illumination, circular track in aluminum, photo-camera fitted to a motorized dolly, a rotating (360°) motorized support fitted to a telescopic arm, an electronic Modified Saschakit Pololu Stepper Controller.
