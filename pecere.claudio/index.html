Project 1: Indoor system for monitoring people's movement

This project aims to create a system that is able to track people movement in indoor space. Track means to measure movement point to point, stay time and people concentrations in critical point of rooms.
The scope of this project is to develop a system that shows this data. This kind of information should be able to:
• Reduce and monitor energy consumption
• Reduce and monitor consumption for heating
• Understand people concentrations for design and modify layout of machinery and product (ex. FabLab, large-scale retail trade)

Project 2: Wearable system for incorrect posture

This second project regards problems and defects in human posture. The idea came up from the boring and expensive process to understand postural problem with actual method and to define right therapy.
The project aims to create a wearable system that shows postural errors while walking, while standing and especially while sitting.
This errors could generate many diseases that affect in a very negative way the day by day life of of children, youth and adults
